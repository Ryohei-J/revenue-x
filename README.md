# RevenueX

個人開発者向けの収益シミュレーター。支出・収入・ユーザー成長を入力し、月次の収支推移をグラフで可視化するWebアプリ。

## Tech Stack

Next.js (App Router) / TypeScript / shadcn/ui / Tailwind CSS / Recharts / next-intl

## Setup

```bash
pnpm install
pnpm dev
```

## Commands

```bash
pnpm dev          # 開発サーバー
pnpm build        # プロダクションビルド
pnpm lint         # ESLint
pnpm test         # ユニットテスト
```
